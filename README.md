# Proyecto Empresario

Este proyecto fue desarrollado como parte de la asignatura **Programación Avanzada (ICI2241)**.  
El objetivo es implementar un sistema de información que permita gestionar emprendedores y bitácoras utilizando **Java** y el **Java Collections Framework**.

---

##  Requisitos previos

Antes de ejecutar este proyecto, asegúrate de tener instalado:

- **Java JDK 11**  
  Descarga desde la página oficial de Oracle o adoptium:  
   https://adoptium.net/

- **Apache NetBeans IDE (12 o superior, compatible con Java 11)**  
  Descarga desde:  
   https://netbeans.apache.org/download/

---

## ⚙ Instalación y configuración en NetBeans

1. Clona este repositorio en tu computadora:  
   ```bash
   git clone https://github.com/JuanGatica18/Proyecto-Empresario.git

2. Abre Apache NetBeans.

3. En el menú superior, selecciona:
  File > Open Project.

4. Busca la carpeta Proyecto-Empresario que clonaste y ábrela.

5. Asegúrate de que el proyecto está configurado para usar Java 11:

    -Clic derecho sobre el proyecto en el panel izquierdo.
    
    -Selecciona Properties.
    
    -En Libraries > Java Platform, elige JDK 11.
    
6. Limpia y construye el proyecto:
    
    -Run > Clean and Build Project.

7. Finalmente, ejecuta el programa:

    -Run > Run Project.

    -También puedes presionar F6.

# ▶Ejecución

Al ejecutar el programa, aparecerá un menú en consola donde se podrán gestionar emprendedores y sus bitácoras.
